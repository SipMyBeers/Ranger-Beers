Battle Drills

React to contact, ambush, IED, and more. Battle drills are graded on every patrol — know them cold so you can execute under sleep deprivation and stress.

Modules
1. React to Contact 2. React to Ambush (Near) 3. React to Ambush (Far) 4. React to IED/UXO 5. Break Contact 6. Consolidation & Reorganization
Module 1
React to Contact

Content coming soon. This module will cover the immediate actions upon receiving direct or indirect fire, including returning fire, seeking cover, and initiating battle drill procedures.

Module 2
React to Ambush (Near)

Content coming soon. This module will cover the immediate assault through the kill zone, actions for soldiers in and out of the kill zone, and the violence of action required to survive a near ambush.

Module 3
React to Ambush (Far)

Content coming soon. This module will cover establishing a support-by-fire position, maneuvering the assault element, and the key differences between near and far ambush responses.

Module 4
React to IED/UXO

Content coming soon. This module will cover the 5 Cs (Confirm, Clear, Cordon, Check, Control), reporting procedures, and actions for the patrol leader.

Module 5
Break Contact

Content coming soon. This module will cover bounding overwatch techniques, smoke employment, and the decision matrix for when to break contact versus assault through.

Module 6
Consolidation & Reorganization

Content coming soon. This module will cover SLLS (Stop, Look, Listen, Smell), ACE reports, LACE reports, EPW handling, and sensitive items accountability after contact.
